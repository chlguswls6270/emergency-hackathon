# 🗞️ The Daily Hacker (더 데일리 해커)

> **"마법 세계의 신문을 펼치듯 새로운 해커톤 특종을 만나보세요."**

단조롭고 기계적인 기존의 IT 행사 플랫폼에서 탈피하여, 영화 해리포터의 '예언자 일보(Daily Prophet)'를 모티브로 한 몰입형 해커톤/팀 빌딩 통합 플랫폼입니다. 사용자는 오래된 마법 신문 환경 속에서 해커톤(특종) 정보를 열람하고, 파티원(동료 마법사)을 구인하며, 명예의 전당에서 우수 개발자들을 확인할 수 있습니다.

---

## 🌟 주요 특징 (Core Features)

### 1. 전역 빈티지/마법 테마 시스템
- **신문 레이아웃 (Masonry)**: CSS 다단 배열(`column-count`)을 활용하여 해커톤 및 구인 기사들이 빈 공간 없이 벽돌처럼(Masonry) 가득 차오르는 언밸런스 지면을 구성했습니다.
- **오버레이 텍스처**: `mix-blend-mode`와 SVG 노이즈 필름을 결합해 브라우저 화면 전체에 수제 양피지와 어두운 비네트 효과를 구현했습니다.
- **Framer Motion 애니메이션**: 안개 속에서 글자가 스며드는 마법적인 트랜지션과 움직이는 초상화(Ken Burns) 효과를 도입해 페이지 전환 간 충돌을 방지했습니다.

### 2. 해커톤 특종 보도 및 하이브리드 참가 연동
- 모집 중이거나 진행 중인 해커톤 정보를 기사 형태로 깊이 있게 시각화합니다.
- `[해커톤 참가 신청하기]` 기능을 통해 가상 백엔드(Local Storage) 기반의 실시간 참가 내역 동기화를 처리합니다.
- 좌측 사이드바 `[나의 참가 내역]` 영역에서 현재 참여 중인 행사를 편안하게 트래킹할 수 있습니다.

### 3. 직군별 팀원/파티 구인구직 (Camp)
- 낡은 전단지 느낌의 구인 게시판에서 프론트엔드, 백엔드, 디자이너, 기획자 등 필요한 파티원을 탐색합니다.
- 조건에 맞는 모집 공고에 클릭 한 번으로 `[합류 제안 보내기]` 트랜잭션을 전송하여 지원자의 정보를 전달합니다.

### 4. 명예의 전당 (Rankings)
- 해커톤이 종료된 후, 제출된 결과물(프로젝트)들을 점수 및 평가 기준으로 나열하는 글로벌 리더보드 및 아카이브(도서관)를 표기합니다.

---

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: React.js, Vite, React Router DOM, Framer Motion
- **Backend (API)**: Python FastAPI 구조 (현 시점엔 오프라인 JSON 모의 데이터 및 Local Storage 하이브리드 동작)
- **Styling**: Vanilla CSS (CSS Variables 패턴 기반의 Custom Scrollbars 및 앤틱 테마)

## 🚀 시작하기 (Getting Started)

아래 명령어를 통해 손쉽게 로컬 환경에서 부엉이(서버)를 날려보낼 수 있습니다.

1. **저장소 내려받기 및 이동**
   ```bash
   cd emergency-hackathon
   cd frontend
   ```

2. **필수 패키지 설치**
   ```bash
   npm install
   ```

3. **개발 서버 실행**
   ```bash
   npm run dev
   ```

터미널에 안내되는 로컬호스트 주소(일반적으로 `http://localhost:5173`)로 접속하여 살아 숨 쉬는 최신 특종을 확인하세요!

---

## 📌 주요 사용 흐름 및 시나리오

1. **최초 접속**: 무한 스크롤 형태의 뉴스 피드에서 쏟아지는 특종(해커톤 개요 및 구인 광고)을 탐색합니다.
2. **이벤트 참가**: 관심 있는 해커톤 상세 페이지에서 `[참가 신청]`을 누르면 즉시 참가자 명부에 로컬 동기화됩니다.
3. **팀 빌딩 및 구인**: 사이트 상단 `Camp(구인 광고)` 탭에서 마음에 드는 동료 모집 공고에 지원 메시지를 발송하거나 직접 팀원 모집을 올립니다.
4. **결과물 제출**: 해커톤 당일 양식에 맞춰 팀 결과물(도출 파일/깃허브)을 입력 폼을 통해 제출하고 리더보드 랭크를 대기합니다.
